# ProtoSMP Modpacks

Official modpack downloads for **ProtoSMP** — a curated Vanilla+ Hermit-style Minecraft SMP.

These modpacks are designed to enhance your experience with:

* better visuals
* smoother performance
* improved quality-of-life
* optional immersive features

All modpacks are **client-side only** and **optional**.

---

# Available Modpacks

## Community Pack (Recommended)

The full ProtoSMP experience.

Includes:

* shaders support
* immersive animations
* improved visuals
* enhanced HUD/UI
* Simple Voice Chat
* Voxy distant terrain (LOD)
* performance optimizations

Recommended for:

* relatively modern gaming PCs
* players who want the intended experience

---

## Community Lite

A lighter version of the Community Pack.

Includes:

* core QoL features
* performance optimizations
* reduced visual effects
* no heavy optional visuals
* animations disbled (can be enabled)

Recommended for:

* lower-end systems
* laptops
* players prioritizing FPS

# Installation

## CurseForge

1. Download the CurseForge `.zip`
2. Open CurseForge
3. Go to "**My Modpacks**"
4. Click "**Import**"
5. Select "**Choose `.zip` file**"
6. Select the CurseForge `.zip`
7. Check the box and choose "**All Files**"
8. Launch when finished
There is an icon if you want to change it!
---

## Prism Launcher **(Recommended)**

1. Download the Prism `.zip`
2. Open Prism Launcher
3. Click "**Add Instance**"
4. Click "**Browse**"
5. Selet the Prism `.zip`
6. Press "**OK**"
7. Launch

---

## Modrinth App (Fastest setup)

1. Download the Modrinth `.mrpack`
2. Run the `.mrpack`
3. Click "**Install anyway**"
4. Launch in library
